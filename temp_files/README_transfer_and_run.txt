# הוראות העברת והרצת הפרויקט במחשב הביתי

1. **העתקת הפרויקט**
   - העתק את כל תיקיית Charles_FocusedSpec (כולל כל התיקיות והקבצים) למחשב הביתי.
   - ודא שהקובץ `.env` עובר יחד עם שאר הקבצים (או צור אותו מחדש עם המפתחות שלך).

2. **התקנת Python**
   - ודא ש-Python 3.10 ומעלה מותקן במחשב הביתי.

3. **התקנת כל החבילות הדרושות**
   - פתח טרמינל/קומנד בנתיב של הפרויקט.
   - הרץ:
     ```
     pip install -r requirements.txt
     ```

4. **בדיקת קובץ .env**
   - ודא שקובץ `.env` מכיל את כל המפתחות (OPENAI_API_KEY, FINNHUB_API_KEY וכו').
   - אם צריך, ערוך אותו עם הערכים שלך.

5. **הרצה ראשונית**
   - להרצת טעינת המסמכים והרצת המערכת:
     ```
     python charles_architect_env/ingest_documents.py
     ```
   - תוכל להריץ כל קובץ אחר בפרויקט באותה צורה.

6. **פתיחה ב-VS Code**
   - פתח את התיקייה ב-VS Code.
   - ודא שה-Interpreter של Python מוגדר נכון (לפי הסביבה בה התקנת את החבילות).

---

אם תיתקל בשגיאות התקנה או הרצה, שלח לי את הודעת השגיאה ואעזור לפתור!
